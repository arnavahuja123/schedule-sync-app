# Schedule Sync

A web app that lets students upload their class schedules and automatically find overlapping classes with friends.

## Features
- Upload schedules
- Compare schedules with friends
- Automatic schedule parsing
- Secure login

## Tech Stack
- React
- Node.js
- PostgreSQL
- Railway

## Installation
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file
4. Run `npm start`
